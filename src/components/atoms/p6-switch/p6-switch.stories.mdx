# Switch

`p6-switch` provide a switch component which allow you to _enable_ or _disable_ a field.

<Story
  name="Default"
  parameters={{
    docs: { disable: true },
    actions: {
      handles: ["p6Submit p6-form"],
    },
  }}
  decorators={[withKnobs]}
>
  {() => {
    const kDisabled = boolean("Disabled", false);
    const kColors = select(
      "Mode",
      ["danger", "warning", "default", "info", "success", "primary"],
      "success"
    );
    const kSizes = select(
      "Size",
      ["small", "normal", "medium", "large"],
      "normal"
    );
    const kText = text("Text", "Switch on by default");
    return `
      <p6-form>
        <p6-switch name="test"
          disabled=${kDisabled}
          mode=${kColors}
          size=${kSizes}
          >${kText}</p6-switch>
          <p6-button class="button is-link">Submit</p6-button>
          <p6-button class="button is-link" type="reset">Reset</p6-button>
      </p6-form>
    `;
  }}
</Story>

### Disabled

The `disabled` _prop_ will keep you from editing it. In the case you are using it inside a form, it won't populate the submit object.

<Preview>
  <Story name="Disabled" height="44px">
    {`
    <p6-switch checked disabled name="switch-disabled">
      Disabled switch
    </p6-switch>
  `}
  </Story>
</Preview>
