# Help

`p6-help` provides an help icon from [Font Awesome](https://fontawesome.com/) with a tooltip

<Story
  name="Default"
  parameters={{ docs: { disable: true } }}
  decorators={[withKnobs]}
>
  {() => {
    const kPosition = select(
      "Position",
      ["top", "right", "bottom", "left"],
      "right"
    );
    const kMode = select(
      "Mode",
      ["default", "info", "warning", "primary", "success", "danger"],
      "default"
    );
    const kTooltip = text("Tooltip", "This is a tooltip!");
    return `<div style="margin:0px 50%; padding-top:50px"><p6-help position="${kPosition}" mode="${kMode}" text="${kTooltip}"></p6-help></div>`;
  }}
</Story>
