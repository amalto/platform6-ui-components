# Link

`p6-link` provides a link. if the href property is set, an anchor tag will be rendered

<Story name="Default" height="42px">
  {() => {
    const anchor = document.createElement('p6-link');
    anchor.innerHTML = 'link';
    anchor.addEventListener('click', action('Click'));
    return anchor;
  }}
</Story>
